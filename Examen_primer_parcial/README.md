## 🚀 Endpoints

### 🔹 GET sin parámetros

* `/` → verificar API
* `/pedidos` → lista de pedidos

---

### 🔹 GET con parámetros

* `/pedido?cliente=` → buscar pedido por cliente

---

### 🔹 POST

* `/pedido` → agregar pedido

---

### 🔹 PUT

* `/pedido` → actualizar pedido

---

### 🔹 DELETE

* `/pedido?cliente=` → eliminar pedido
